¿Qué es el Diseño Responsivo?

Es crear un sitio web que se adapte a diferentes dispositivos proporcionalmente, manteniendo su estructura y orden.

¿Es fundamental una Web Responsiva?

Si, 60% de los usuarios desconfiarán de una empresa con un sitio web que no es responsivo.
